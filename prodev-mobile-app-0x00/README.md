# First Mobile App with Expo Router

## Objective
Set up the first mobile application using the Expo Router template and understand the project structure.

## Scaffolding Steps
1. Navigated to the parent project directory.
2. Initialized a new Expo app using the Expo Router template.
3. Reviewed the generated folder and file structure.
4. Modified the home screen content.
5. Ran the application using Expo Go.

## Reset Project Observation
Running the reset-project command removed the default example content and restored a clean project state while keeping the core Expo Router structure intact.
