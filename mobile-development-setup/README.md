# Mobile Development Setup

## Objective
Set up the mobile development environment using Expo Go for testing React Native applications on a physical device.

## Tools Used
- Node.js (LTS)
- Visual Studio Code
- Expo CLI
- Expo Go

## Setup Steps
1. Visited https://expo.dev/go
2. Installed Expo Go on a physical mobile device.
3. Logged into Expo Go successfully.

## Observations
Expo Go allows testing React Native apps directly on a physical device without using emulators.
