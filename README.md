# Mobile Development Setup

## Objective
Set up the mobile development environment using Expo Go for testing React Native applications on a physical device.

## Tools Used
- Node.js (LTS)
- Visual Studio Code
- Expo CLI
- Expo Go (Android / iOS)

## Setup Steps
1. Visited https://expo.dev/go
2. Selected the latest Expo SDK version.
3. Installed Expo Go from the app store on my mobile device.
4. Opened the Expo Go app and logged in successfully.

## Observations and Challenges
- Installation was straightforward.
- Expo Go allows testing applications directly on a physical device without setting up emulators.
- No major challenges encountered during setup.
